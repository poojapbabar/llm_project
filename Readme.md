## Installation

1. **Create a Conda environment:**
    ```bash
    conda create --name myenv python=3.8
    ```

2. **Activate the Conda environment:**

    - On Windows:
        ```bash
        conda activate myenv
        ```

    - On macOS and Linux:
        ```bash
        source activate myenv
        ```

3. **Install Flask:**
    ```bash
    conda install flask
    ```

4. **Run the Flask application:**
    ```bash
    python app.py
    ```
